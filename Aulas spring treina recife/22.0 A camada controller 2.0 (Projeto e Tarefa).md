Os controllers de Projeto e Tarefa também ganharam novos endpoints na v2.0, seguindo o mesmo padrão do `UsuarioController`.

---

## ProjetoController

```Java
@RestController
@RequestMapping(value = "/projetos")
public class ProjetoController {

    @Autowired
    private ProjetoService projetoService;

    @PostMapping
    public ResponseEntity<DadosProjeto> cadastrar(@RequestBody Projeto projeto) {
        return ResponseEntity.status(HttpStatus.CREATED)
            .body(projetoService.salvarProjeto(projeto));
    }

    @GetMapping
    public ResponseEntity<List<DadosProjeto>> listar() {
        return ResponseEntity.ok(projetoService.listarProjetos());
    }

    @GetMapping("/{id}")
    public ResponseEntity<DadosProjeto> buscarPeloId(@PathVariable Long id) {
        DadosProjeto projeto = projetoService.buscarProjeto(id);
        if (Objects.isNull(projeto)) return ResponseEntity.notFound().build();
        return ResponseEntity.ok(projeto);
    }

    @GetMapping("/status/{status}")
    public ResponseEntity<List<DadosProjeto>> buscarPeloStatus(@PathVariable StatusProjeto status) {
        return ResponseEntity.ok(projetoService.buscarPeloStatus(status));
    }

    @GetMapping("/responsavel/{usuarioRespId}")
    public ResponseEntity<List<DadosProjeto>> buscarPeloResponsavel(@PathVariable Long usuarioRespId) {
        return ResponseEntity.ok(projetoService.buscarPeloResponsavel(usuarioRespId));
    }

    @PutMapping("/{id}")
    public ResponseEntity<DadosProjeto> atualizar(@PathVariable Long id, @RequestBody Projeto dadosAtualizados) {
        DadosProjeto projeto = projetoService.buscarProjeto(id);
        if (Objects.isNull(projeto)) return ResponseEntity.notFound().build();
        dadosAtualizados.setId(id);
        return ResponseEntity.ok(projetoService.salvarProjeto(dadosAtualizados));
    }

    @DeleteMapping("/{id}")
    public ResponseEntity<Void> excluir(@PathVariable Long id) {
        DadosProjeto projeto = projetoService.buscarProjeto(id);
        if (Objects.isNull(projeto)) return ResponseEntity.notFound().build();
        projetoService.excluirProjeto(id);
        return ResponseEntity.noContent().build();
    }
}
```

### Todos os endpoints do ProjetoController

| Método | URL | O que faz |
|---|---|---|
| `POST` | `/projetos` | Cria um projeto |
| `GET` | `/projetos` | Lista todos os projetos |
| `GET` | `/projetos/{id}` | Busca pelo ID |
| `GET` | `/projetos/status/{status}` | Filtra por status |
| `GET` | `/projetos/responsavel/{id}` | Busca projetos de um responsável |
| `PUT` | `/projetos/{id}` | Atualiza um projeto |
| `DELETE` | `/projetos/{id}` | Remove um projeto |

---

> 💡 **Observação:** repara que tanto `buscarProjeto()` quanto `buscarTarefa()` agora retornam `DadosProjeto` e `DadosTarefa` respectivamente — não mais a entidade crua. A verificação de `null` continua funcionando da mesma forma.
