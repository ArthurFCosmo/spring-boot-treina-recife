
---

O Springboot é um framework do Java utilizado para a construção de API's. API's são aplicações que servem para integrar sistemas uns aos outros. Elas recebem dados, tratam, e devolvem, assim como um carteiro.

O Springboot é poderoso pois consegue integrar bancos de dados SQL com frontend através de API's feitas em Java. Ele serve como um facilitador, pois essas integrações exigem muita "burocracia", fazendo com que seja muito mais prático, rápido e organizado o desenvolvimento de API's.
