## Criando um Banco de Dados no SQL

Nosso primeiro passo vai ser criar um banco de dados.

No workbench, acessaremos qualquer conexão com o MySQL, daremos um botão direito na barra lateral e "create schema", depois é so dar ok em tudo.

## Conectando o Banco de dados na nossa API

Para conectar nosso BD, deveremos ir no arquivo `application.properties` na pasta resources.

Nessa pasta, colaremos um bloco de código que tiraremos da documentação do Springboot, disponível em https://spring.io/guides/gs/accessing-data-mysql .

Ficando assim:

```MySQL
# É utilizado para transformar classes do projeto em tabelas no BD
spring.jpa.hibernate.ddl-auto=update

# Esse comando serve para exibir o SQL que está sendo rodado por baixo dos panos (bom para debugar)
spring.jpa.show-sql=true 

# Dados de acesso ao BD
spring.datasource.url=jdbc:mysql://${MYSQL_HOST:localhost}:3306/mydatabase spring.datasource.username=
spring.datasource.password=
```

No campo de username coloque o user do MySQL e a senha no campo passowrd.
